# SentinelPhishFeed

SentinelPhishFeed is a continuously updated, consolidated collection of
high-signal threat indicators intended for defensive security use.

## What this repository provides
- Plaintext threat indicators (IPs, URLs, domains, hashes)
- Periodic refresh on a fixed schedule
- Deduplicated, normalized data
- No per-source attribution

## Intended use
- Detection and prevention
- Threat hunting
- Research and analysis

This repository aggregates data from multiple open sources and is
provided **as-is**, without warranty or guarantees.

## Update cadence
- Automatically refreshed every few hours
